# PDF Forge Clone (iLovePDF Style)

A Next.js app inspired by iLovePDF with a large tools catalog UI and working server-side PDF processing routes.
You can try it here:
👉 https://ilovepdfeditor.netlify.app/

## Live Working Tools

- Merge PDF
- Split PDF (page ranges)
- Compress PDF (structure optimization)
- Organize PDF (custom page order)
- Rotate PDF
- Watermark PDF (text, opacity, angle, size)
- Add Page Numbers
- Crop PDF
- JPG/PNG to PDF
- PDF to JPG
- Word to PDF (DOCX)
- PDF to Word (text-focused DOCX)
- HTML to PDF
- Unlock PDF (rasterized output)

## Tech Stack

- Next.js 15 (App Router)
- React 19 + TypeScript
- Tailwind CSS
- `pdf-lib` for PDF processing

## Local Run

```bash
npm install
npm run dev
```

Open: `http://localhost:3000`

## Production Build Check

```bash
npm run lint
npm run build
```
