# 👑 BOSS.AI — Your Personal Command Center

A sleek, voice-powered AI assistant that calls you Boss and does whatever you command.

## Features
- 🎙️ Voice recognition (speak in any language)
- 🔊 Text-to-speech (AI talks back)
- 🌐 Multilingual (English, Hindi, Bengali, Spanish, French, Arabic, Chinese, Japanese)
- ⚡ Quick command shortcuts
- 🧠 Conversation memory (last 40 messages)
- 📱 Mobile responsive
- 🔥 Cyberpunk gold UI

## Deploy to Netlify (Free)

### Option 1: Drag & Drop
1. Go to https://app.netlify.com
2. Sign up / log in (free)
3. Drag the entire project folder to the deploy zone
4. Your site is LIVE instantly!

### Option 2: GitHub + Netlify
1. Push this folder to a GitHub repo
2. Connect repo to Netlify
3. Set publish directory to `.`
4. Deploy!

## Usage
- **Voice**: Click the gold orb or 🎙️ mic button → speak → AI responds
- **Text**: Type in the input box and press ENTER or SEND
- **Language**: Click a language flag to switch voice recognition language
- **Quick Commands**: Use the side panel shortcuts
- **Voice Settings**: Change AI voice and speed in the Voice panel

## Powered by
- Google Gemini 2.0 Flash API
- Web Speech API (built into Chrome/Edge)
- 100% frontend — no backend needed!

## Notes
- Works best in Chrome or Edge (best voice support)
- Requires microphone permission for voice commands
- API key is embedded — for production, consider a backend proxy
